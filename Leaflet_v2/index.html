Welcome to Argentine Football Clubs Website

Experience an interactive map below of football clubs in Argentina, there are a list of instructions below the map on how to use the Map. The map contains datasets of each club which can be viewed in the map, Clubs API If you are enjoying the content of the website, then please consider supporting us by donating to our Patreon so quality content can be created in the future. Have a look at other content we have created on our Github Want to get in touch with us? What should we create next? send us a cheeky dm on LinkedIn!

Instructions

  1. Zoom in and out using your mouse or for more control use the zoom in and out buttons on the top left of the map
  2. To change map, click the white box in the top right corner of the map and select by clicking on the box next to the map you wish to use
  3. Drag your mouse across the map or use arrow keys in the direction you want
  4. Click the badges on the map to see the name of the football club and to click off the popup then either click on the map or use the X button in the top right of the popup
  5. To see the same information that is on the map, please click here
