About

Who we are and what are we trying to achieve?

We are both junior software developers based in Scotland who graduated from CodeClan (SQA accredited digital skills academy). We pride ourselves on making our content accessible and responsive to ensure the user experience is good. We are making an attempt to increase our self learning by creating projects such as this one, to make better all-round Software Developers and to hopefully gain employment as a Junior Fronted Software Developer.

Why this Project and what we learned?

The Map was the only feature originally designed for this project until we both agreed to make it more accessible to add in a clubs page so people who could not access the map. We then took the opportunity on as a project to test our fronted software development skills such as JavaScript, HTML, CSS and gain familiarity with Leaflet which was used to create this website. Little aspects such as being able to know where to place components on the web page has shown how far our CSS has came along, knowing the semantic markup (HTML) will help browsers acknowledge what part of the webpage is what. Improving our JavaScript by getting more familiar with event listeners, arrays and methods that allow for a functional search bar to be incorporated as well as gaining knowledge on modal windows to have another element of user interface.

A Big Thank You

We want to give a massive thanks to Wikipedia, Google, Youtube and Listal for data, images, videos and translating information. None of this project would have been possible without them.
